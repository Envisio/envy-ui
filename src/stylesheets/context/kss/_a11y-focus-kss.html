1. No Context (default focus)

Standard focus styling - light blue focus ring.

2. A11y Context: data-env-ui-context="publish"

High-contrast black focus on keyboard only (:focus-visible). Click does NOT show focus.

3. A11y Context + Focus Policy: data-env-ui-focus-policy="all"

High-contrast black focus on all interactions (keyboard + mouse click).

4. Custom Color: --env-ui-a11y-focus-color: #0066cc

Override default black focus with custom color using CSS variable.
