1. No Context (default focus)

Standard soft glow focus ring.

2. data-env-ui-context="publish"

High-contrast black focus on keyboard only. Click does NOT show focus.

3. data-env-ui-context="publish" data-env-ui-focus-policy="all"

High-contrast black focus on all interactions (keyboard + mouse).

4. Custom Color: --env-ui-a11y-focus-color: #0066cc

Override default black focus with custom color using CSS variable.
